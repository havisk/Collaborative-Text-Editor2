# Collaborative Text Editor.

- work with others on the same document with realtime update.


## Tech Used

- Meteor
- Session
- mizzao:sharejs-codemirror
- Bootstrap
- MongoDB

## Usage

- Clone this repo
- Run `meteor` to start the server 
- go to localhost:3000 in browser of choice.
